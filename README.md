Input the `start` and `end`, and it will output digits from the `start`th to the `end`th to output/output.txt .

e.g.

in Windows
```shell
> mkdir build
> mkdir output
> make
> .\build\pi_hex.exe
start=1
end=10
> cat .\output\output.txt
243f6a8885
```

in linux
```bash
$ mkdir build
$ mkdir output
$ make
$ .\build\pi_hex
start=1
end=10
$ cat output/output.txt
243f6a8885
```
